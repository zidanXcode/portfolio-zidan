Project One: WhatsApp Bot

Sebuah bot otomatis berbasis JavaScript dan node.js untuk platform WhatsApp, dirancang untuk interaksi cepat, otomatisasi tugas, dan pengelolaan pengguna berbasis database.

Fitur Utama

  • Auto-reply untuk kata kunci tertentu
  • Database pengguna menggunakan file JSON atau MongoDB
  • Perintah dinamis: !menu, !help, !owner, dll.
  • Dukungan media: gambar, audio, dokumen
  • Deteksi pesan grup vs personal
  • Logger aktivitas pengguna

Teknologi yang Digunakan

JavaScript Node.js whatsapp-web.js MongoDB / JSON
