Project Two: UI/UX Web Design

Proyek ini berfokus pada pembuatan website portofolio modern dan interaktif menggunakan HTML, Tailwind CSS, dan JavaScript. Tujuannya adalah menciptakan tampilan profesional, responsif di semua perangkat, dan pengalaman pengguna yang menyenangkan.

  • Responsif: Desain adaptif untuk mobile & desktop.
  • Transisi Halus: Animasi AOS saat scroll dan hover elegan.
  • Mode Gelap: Nyaman di mata, gaya modern.
  • Struktur UI Jelas: Elemen konsisten dan intuitif.
